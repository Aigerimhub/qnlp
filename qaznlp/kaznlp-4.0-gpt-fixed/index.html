🇰🇿 KazNLP

Kazakh Natural Language Processing Toolkit:

  • Morphological analysis
  • POS tagging with KazBERT + CRF
  • Inflection and generation
  • JSONL-based annotation tool
  • Corpus export (CoNLL-U, TSV)
